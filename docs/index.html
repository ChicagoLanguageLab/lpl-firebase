(July 2012, Erik Weitnauer) The html-plugin will load and display an arbitrary number of html pages. To proceed to the next, the user might either press a button on the page or a specific key. Afterwards, the page get hidden and the plugin will wait of a specified time before it proceeds.

Palmer, S. (1977). Hierarchical Structure in Perceptual Representation. Cognitive Psychology, 9, 441.
